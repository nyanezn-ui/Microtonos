# 🎵 Simulador de Microtonos en Python

¡Bienvenido al **Simulador de Microtonos**! Este es un script interactivo en Python diseñado para entornos Windows que permite experimentar con la generación de frecuencias sonoras y la gestión de recursos de audio simulados (microtonos) a través de la consola.

El programa utiliza la biblioteca nativa `winsound` para hacer vibrar los altavoces de tu placa base o auriculares con frecuencias y duraciones específicas.

---

## 🚀 Características

* **Gestión de recursos:** Controla un límite máximo de 25 microtonos.
* **Reproducción secuencial:** Genera una secuencia de tonos predefinidos basados en frecuencias y duraciones musicales variables.
* **Monitoreo en tiempo real:** Revisa cuántos microtonos están activos "haciendo vibrar el ambiente".
* **Sistema interactivo:** Menú dinámico con validación de errores para evitar fallos por ingresos de datos incorrectos.

---

## 🛠️ Requisitos del Sistema

* **Sistema Operativo:** Windows (requerido para la librería `winsound`).
* **Python:** Versión 3.x o superior.
* **Hardware:** Altavoces o auriculares habilitados.
